Git Commands & Description

1. git init

To initialize git in our project folder.

2. git status

To check or show status of tracked and untracked files from git cache

3. git add filename

To add single file to git cache which are untracked

4. git add .

to add all untracked files to git cache

5. git rm --cached filename

to remove the last added untracked files from the git cache

6. git commit -m "message"

generates the committed id with commit message to make ready to push
